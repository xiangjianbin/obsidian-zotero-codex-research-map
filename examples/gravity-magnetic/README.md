# 重力磁力反演实例版

这个版本展示如何把“重力磁力反演”组织成科学问题树。

## 主线

1. Li & Oldenburg 1996/1998：传统重力、磁力 3D 反演基线。
2. Gallardo & Meju 2004、Gallardo 2007：交叉梯度源头。
3. Fregoso & Gallardo 2009：重磁交叉梯度联合反演。
4. Vatankhah 2020、Liu 2023：交叉梯度效率和 Lp 正则。
5. Zhou 2023、Bai 2024、Fang 2025：深度学习重磁联合反演主线。
6. Gupta 2026：生成式/后验近似方向。
7. Jessell 2022 / Noddyverse：合成数据和泛化问题。

## 最重要的科学挑战

- 非唯一性与病态性。
- 深度分辨率衰减。
- 磁化方向与剩磁影响。
- 结构一致性假设。
- 多源权重与尺度耦合。
- 合成到实测域迁移。
- 不确定性量化。

## 最适合 ML 的切入点

首选不是“端到端 CNN 反演”，而是：

```text
学习式软结构耦合
= 判断哪里应该共享结构，哪里应该允许重力源和磁力源解耦。
```

其次：

- 合成到实测域迁移。
- 生成式后验近似。
- 深度感知先验。
- 传统求解器加速和混合反演。
