# 重磁反演的机器学习机会

## 优先级 1：学习式软结构耦合

研究问题：重力源和磁力源在什么位置应该共享边界，在什么位置应该允许错位或解耦？

传统基线：交叉梯度联合反演。

ML 形式：

- 多模态特征对齐。
- 局部结构相似性概率图。
- 空间可变耦合权重。
- 独立/联合反演区域自动切换。

必须检查：

- 重力和磁力正演残差。
- 非同源异常体测试。
- 训练分布外测试。
- 与传统交叉梯度公平比较。

## 优先级 2：合成到实测迁移

研究问题：合成数据训练出的网络如何迁移到真实矿区？

需要记录：

- 地质体几何分布。
- 物性范围。
- 噪声模型。
- 测网设置。
- 区域场处理。
- 钻孔或地质约束。

## 优先级 3：生成式后验近似

研究问题：能否快速给出多个等效模型，而不是单一反演结果？

可能方法：

- normalizing flow
- diffusion model
- rectified flow
- VAE latent prior

必须检查：

- 每个样本是否能正演回观测。
- 不确定性是否校准。
- 多解是否有地质意义。
