# 通用科研问题树模板版

把任意领域文献组织成问题树，可以套用这个结构：

```text
研究主题
├─ 固有科学困难
├─ 传统方法基线
├─ 方法改进路线
├─ 数据与验证问题
├─ 机器学习/新方法机会
└─ 未解决缺口
```

## 适用领域

- 地球物理反演。
- 材料科学。
- 医学影像。
- 气候模型。
- 遥感。
- 任何需要系统综述和方法对比的研究方向。

## 基本产物

- `00 总览.md`
- `00-问题树.md`
- `01-文献矩阵.md`
- `02-机器学习可切入问题.md`
- `问题节点/`
- `Zotero/文献卡片/`

## 最小工作流

1. 定义 5-12 个问题节点。
2. 每篇文献挂到 1-3 个节点。
3. 记录解决程度和未解决缺口。
4. 用 Dataview 自动生成矩阵。
5. 定期让 Codex 检查重复、断链和空字段。
